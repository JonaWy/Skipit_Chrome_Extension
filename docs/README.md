# SkipIt Documentation

This directory contains public documentation for the SkipIt Chrome Extension, including the Privacy Policy.

## GitHub Pages Setup

To enable GitHub Pages for this repository:

1. Go to your repository settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose the `main` (or `master`) branch
5. Select the `/docs` folder
6. Click "Save"

Your Privacy Policy will be available at:
`https://yourusername.github.io/repository-name/privacy-policy.html`
