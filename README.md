## About

**SkipIt** is a free Chrome extension that automatically skips intros, recaps, and credits on popular streaming platforms and gives you precise control over video playback speed on any website.

Stop wasting time clicking "Skip Intro" ‚Äì let SkipIt handle it for you while you enjoy your content.

## Features

### Auto-Skip
- **Skip Intro** ‚Äì Automatically clicks "Skip Intro" buttons
- **Skip Recap** ‚Äì Skips "Skip Recap" on supported platforms
- **Skip Credits** ‚Äì Jumps past end credits to the next episode
- **Multi-language** ‚Äì Works with skip buttons in English, German, Spanish, French, and more

### Speed Control
- **Range** ‚Äì Adjust playback from 0.25x to 4.0x
- **8 Presets** ‚Äì Customizable quick-access buttons
- **Universal** ‚Äì Works on any HTML5 video player (YouTube, Vimeo, educational sites, etc.)
- **Per-Site Memory** ‚Äì Remembers your preferred speed for each domain

### Additional Features
- **Statistics** ‚Äì Track how many intros you've skipped and time saved
- **Dark Mode** ‚Äì Easy on the eyes for night-time viewing
- **Cross-Device Sync** ‚Äì Settings sync via your Chrome profile
- **Privacy-First** ‚Äì No data collection, no analytics, no trackers

### Manual Installation (Developer Mode)

1. Download or clone this repository
   ```bash
   git clone https://github.com/Jonawy/Skipit_Chrome_Extension.git
   ```
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked**
5. Select the cloned folder

## Keyboard Shortcuts

| Shortcut | Action |
